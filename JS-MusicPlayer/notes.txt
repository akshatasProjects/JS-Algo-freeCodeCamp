### Algorithm

- step 1: create a variables and DOM through its ID and a audio HTML element
- step 2: Create a variable "allSongs" array of objects 10 having all song details
- step 3: create a Object assigned with allsongs using spread operator
- step 4: create an arrow function "renderSongs" with map() creating <li>element and button element
- step 5: Assign songsHTML to the playlistSongs to display
- step 6: call renderSongs function with userData?.songs
- step 7: create a variable called sortSongs for arrow function.
- step 8: inside an arrow function userData?.songs assign sort function
- step 9: inside sort() add a callback function
- step 10: inside a callback pass 2 parameters a, b and check condition
- step 11: if(a>b) return -1, b>a return 1, return 0
- step 12: at the end return userData?.songs
- step 13: call the render function call sort function.

----------------------------------------------

1. "playlist-songs" ID - is the UL list
2. Play and Pause are the button under div class "player-content"

image and buttons another is for the song list.

const sortSongs = () => {
  userData?.songs.sort((a, b) => {
    if (a.title < b.title) {
      return -1;
    }
    if (a.title > b.title) {
      return 1;
    }
    return userData?.songs;
  });
};
